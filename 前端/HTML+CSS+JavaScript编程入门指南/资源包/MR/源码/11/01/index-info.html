如图所示，四张图片分别实现旋转，伸缩，平移，倾斜的效果。

应用transform属性的rotate函数可以实现2D旋转。应用transform属性的函数可以实现缩放。应用transform属性的translate函数可以实现2D平移。应用transform属性的skew函数可以实现倾斜。
